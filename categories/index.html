# 功函数的定义

功函数（Work Function）是指将一个电子从材料的费米能级（Fermi level）移动到真空能级（Vacuum level）所需的最小能量。在催化研究中，功函数是表征材料电子结构的重要物理量，常用于分析表面反应活性、电子转移能力和催化选择性。其数学表达式为：

Φ = Vvacuum − EF

其中：

  • Vvacuum：真空区域的电势，即远离材料表面时电势趋于平坦的区域；
  • EF：材料的费米能级；
  • Φ：功函数，单位通常为电子伏特（eV）。

# new命令原理

使用new命令时，它后面的函数依次执行下面的步骤：

  1. 创建一个空对象，作为将要返回的实例对象。
  2. 将这个空对象的原型，指向构造函数的prototype属性。
  3. 将这个空对象赋值给函数内部的this关键字。
  4. 开始执行构造函数内部的代码。
  5. 如果构造函数内有返回值且为对象类型，则返回该对象，否则返回上面创建的实例对象。
