# 计算中常见交换-相关泛函的选取与设置指南

密度泛函理论（Density Functional Theory, DFT）是当前材料模拟和分子模拟中最广泛使用的计算方法之一。而交换-相关能泛函（Exchange-Correlation Functional, XC Functional）的选择，则直接决定了计算的精度、适用性以及物理可靠性，因此本文章整理了常用 XC 泛函的理论基础、适用范围以及如何在主流程序中设置。

# 各类常用泛函详解

# 🔹 LDA（Local Density Approximation）

  • 只依赖密度本身，基于均匀电子气模型；
  • 快速、计算量小，但通常低估晶格常数和能隙；
  • 适合金属体系或大体积测试。

代表：VWN、CA（Ceperley–Alder）

# 🔹 GGA（Generalized Gradient Approximation）

  • 引入密度梯度修正，改善结构与能量描述；
  • PBE 是最常用的默认泛函；
  • RPBE 修正吸附过强的问题；
  • BLYP 常用于有机分子模拟。

适用：结构优化、表面吸附、稳定性分析等

# 🔹 meta-GGA（如 SCAN）

  • 加入动能密度项，能更好描述非局域效应；
  • SCAN 是无经验型泛函，适用范围广；
  • M06-L 对分子反应更有效，TPSS 是过渡型函数。

优点：在不显著增加计算成本的前提下接近杂化泛函精度。

# 🔹 杂化泛函（Hybrid Functionals）

  • 将一部分 HF（Hartree-Fock）交换与 GGA 结合，提高电子结构精度；
  • HSE06 局域化 HF 交换，适合固体材料；
  • B3LYP 为经典分子化学泛函，广泛用于小分子；
  • PBE0 为全局混合泛函，适用于中等体系。

适用：带隙计算、缺陷能、光学性质、电荷分布、反应路径分析等。

# 🔹 RPA 与 Double-Hybrid 泛函

  • 描述电子关联更准确，理论极限；
  • 用于小体系的能量排序、基准研究；
  • 计算资源极大，不适合大体系。

# 各程序中的设置方式

# ✅ VASP

泛函 设置示例
PBE GGA = PE（默认）
RPBE GGA = RP
PBEsol GGA = PS
SCAN METAGGA = SCAN + LASPH = .TRUE.
HSE06 LHFCALC = .TRUE.，HFSCREEN = 0.2，AEXX = 0.25
PBE+U LDAU = .TRUE.，配合 LDAUL, LDAUU, LDAUJ
B3LYP（近似） GGA = B3, AEXX = 0.20, AGGAX = 0.72, AGGAC = 0.81, ALDAC = 0.19

💡 注：

  • VASP 不原生支持 B3LYP；
  • 上述设置为手动近似模拟其交换-相关结构（不能等效于 Gaussian 中的 B3LYP）；
  • 建议如需准确使用 B3LYP，请选用支持该函数的程序如 Gaussian、ORCA、Q-Chem 等；
  • 固体体系推荐优先使用 HSE06 / PBE0。

# ✅ Gaussian

泛函 使用方式（头文件）
B3LYP # B3LYP/6-31G(d)
M06-2X # M06-2X/def2TZVP
CAM-B3LYP # CAM-B3LYP/6-311++G**
ωB97XD 含长程修正 + 色散
DFT-D3 加入 EmpiricalDispersion=GD3

# ✅ ORCA

  • 设置泛函：! B3LYP def2-TZVP TightSCF
  • 加入色散修正：%method Dispersion D3BJ end
  • 支持 HSE06、PBE0、B97-3c 等多种混合泛函
