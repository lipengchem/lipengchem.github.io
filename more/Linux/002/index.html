    编辑器的使用

    在命令行环境中，高效使用文本编辑器是科研、开发与系统管理的基本技能之一。本文将简要介绍常用编辑器（如 Vim、Nano、Emacs）的基本用法。

    # Vim的使用

    在命令行环境中，vim 是一款功能强大且广泛使用的文本编辑器。相比于 nano 的简单易用，vim 更强调效率与可扩展性，是许多程序员和科研工作者的首选。

    # 1. Vim 的三种模式

    模式名称 功能 进入方式
    正常模式 默认模式，执行命令 打开文件即为此模式，或按 Esc
    插入模式 编辑文本内容 i, a, o 等命令进入
    命令模式 保存、退出、查找等操作 正常模式下输入 :

    # 2. 常用命令总览

    • 启动与退出
    vi filename          # 打开文件
:q                   # 退出（未修改时）
:q!                  # 强制退出（不保存）
:w                   # 保存
:wq 或 :x            # 保存并退出
ZZ                   # 保存并退出（快捷）
