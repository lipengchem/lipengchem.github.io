VASP输出文件

在使用 VASP 进行第一性原理计算后，会生成大量输出文件。这些文件中蕴含了能量、力、结构、电荷密度、波函数等各种信息，是后续分析的基础。本篇将系统介绍每个输出文件的用途、格式、提取方法及后处理建议，方便日常查阅。

# 核心输出文件说明

文件名 简要说明
OUTCAR 主日志文件，包含能量、力、应力、电荷、磁矩、投影态等最全信息
OSZICAR 能量/结构收敛简要信息，适合快速检查收敛情况
CONTCAR 当前优化后的结构（POSCAR 格式），可作为后续计算的输入
vasprun.xml XML 格式的完整输出文件，适用于 VASPKIT、pymatgen、VESTA 等
EIGENVAL 能带本征值，含所有 k 点与能级
DOSCAR 总态密度 + 分态密度（需配合 LORBIT）
CHGCAR 电荷密度 + 核位置，用于差分电荷、Bader 分析
LOCPOT 局域电势，用于功函数计算
WAVECAR 波函数文件（很大），用于 Wannier、轨道图等后处理
IBZKPT 实际使用的 k 点路径信息
REPORT 较旧的辅助信息文件（现代版本已不常用）

# 输出文件详细介绍

# 📘 OUTCAR

包含所有详细输出信息，是分析收敛性、力、应力、电荷、磁矩、杂化泛函设置等的重要文件。你可以在其中查到：

  • 总能量与每步电子迭代信息
  • 离子步优化数据（位移、力）
  • 投影态（PARCHG, PROCAR）设置与结果
  • 杂化泛函（如 HSE）收敛情况

适用于精细分析，但文件通常较大。

# 📘 OSZICAR

简化版本日志，仅列出每步电子优化或结构优化的能量与磁矩（若设置 ISPIN = 2）。便于快速查看能量收敛趋势。

# 📘 CONTCAR

结构优化后的结构文件，格式与 POSCAR 相同，包含当前结构（坐标、晶格、固定标志等）。需要注意：

  • VASP 不会自动更新 POSCAR
  • 若要继续计算，需手动复制 cp CONTCAR POSCAR

# 📘 vasprun.xml

机器可读的全量输出文件，包含所有结构、能量、电荷、DOS、能带等信息。适用于：

  • VASPKIT、pymatgen、ASE 分析
  • 自动提取结构参数、体积、应力张量等

虽然大，但功能强大，建议保留。

# 📘 EIGENVAL

能带本征值文件，包含每个 k 点上的所有能级信息。

  • 每一行对应一个 k 点
  • 常与 OUTCAR 联合使用以获取费米能级位置

# 📘 DOSCAR

态密度文件，包含：

  • 总态密度（DOS）
  • 投影态密度（PDOS），需开启 LORBIT ≥ 10 和 RWIGS

可配合 VASPKIT、sumo 进行绘图。

# 📘 CHGCAR

包含电子密度和离子核位置信息，是差分电荷图、Bader 分析、VESTA 画图的输入。

  • 与 AECCAR 配合使用可分离赝电荷
  • 数据为网格格式

# 📘 CHG

CHG 文件保存的是系统的静态电子密度分布（仅电荷密度部分），格式与 CHGCAR 相似，但不包含离子位置信息。

  • 由 ICHARG = 11 或 ICHARG = 0 等计算中生成；

  • 常用于静态计算或 DOS / 能带计算前的预读取；

  • 与 CHGCAR 相比，它更“轻量”一些，但不能单独用于可视化；

  • 与 CHGCAR 的区别

特征 CHG CHGCAR
电荷密度 ✅ 包含 ✅ 包含
离子坐标信息 ❌ 不包含 ✅ 包含
可用于 VESTA ❌ 不推荐 ✅ 可直接可视化
与 AECCAR 合作 ❌ 无法使用 ✅ 可配合做差分电荷分析
推荐用途 供下一步读取电荷 可视化 / 后处理 / Bader 分析

💡 如果只是为了保存/读取前一步的电子密度，用 CHG 即可；
💡 如果要做电荷图、功函数、差分密度分析，应使用 CHGCAR。

  • 常见用法

在做静态计算（比如能带 / DOS）时，通常可以：

  1. 从结构优化后的目录中拷贝 CHG：

     cp ../opt/CHG .
    1

  2. 设置 INCAR：

  ICHARG = 11    # 读取已有电荷密度
1

# 📘 LOCPOT

局域电势文件，包含静电势的空间分布，常用于：

  • 计算功函数（截取真空面）
  • 可用 VESTA 观察表面势

# 📘 WAVECAR

包含波函数信息（Plane-wave 展开系数），可用于：

  • Wannier90 接口
  • STM 模拟
  • 轨道分析

⚠️ 通常很大，不建议长期保留，除非有明确波函数分析任务。

# 📘 IBZKPT

实际使用的 k 点路径文件，VASP 会根据 KPOINTS 自动生成。

  • 可用来确认采样点是否正确，尤其在高对称路径能带中

# 📘 PROCAR（可选）

  • 投影态密度分析文件
  • 含轨道分量、原子贡献
  • 需设置 LORBIT = 10~12 生成

# 📘 ELFCAR（可选）

  • 电子局域化函数数据（Electron Localization Function）
  • 用于分析共价键强度、离域程度
  • 可用 VESTA 进行可视化

# 📘 PARCHG（可选）

  • 态选择电荷密度
  • 可用于绘制特定轨道（如导带底/价带顶）的空间分布
  • 设置 LPARD = .TRUE. 并指定能级范围

# 📘 AECCAR0/1/2（可选）

  • 分别表示：核电荷密度（AECCAR0）、赝电荷密度（AECCAR1）、总电荷密度（AECCAR2）
  • 与 CHGCAR 配合可做差分密度图、Bader 电荷分析

# 📘 XDATCAR（可选）

  • 包含每一步结构变化轨迹
  • 用于分子动力学（MD）分析或 NEB 路径提取

# 输出文件用途对应表

分析目标 所需输出文件组合
结构优化结果 CONTCAR, OUTCAR, OSZICAR
功函数计算 LOCPOT, CONTCAR
能带结构分析 EIGENVAL, OUTCAR
态密度（DOS） DOSCAR, vasprun.xml, PROCAR（PDOS）
差分电荷密度 CHGCAR, AECCAR0, AECCAR2, CONTCAR
Bader 电荷分析 CHGCAR, AECCAR0, AECCAR2
轨道投影分析 PROCAR, PARCHG, WAVECAR
分子动力学分析 XDATCAR, OUTCAR

# 小提示建议

  • 💡 只做结构优化：可删除 WAVECAR, CHGCAR 节省空间；
  • 💡 功函数分析：保留 LOCPOT, CONTCAR；
  • 💡 波函数后处理：WAVECAR 必不可少，但文件很大；
  • 💡 下一步计算：别忘记 cp CONTCAR POSCAR！

# 后处理工具推荐

工具 功能
VESTA 结构、电荷密度、ELF、LOCPOT 可视化
VASPKIT DOS、能带、功函数、轨道分析等
pymatgen 提取结构、电荷、能量数据
VasprunParser 快速读写 vasprun.xml 文件
sumo DOS、能带自动化绘图工具
