# 功函数的定义

功函数（Work Function）是指将一个电子从材料的费米能级（Fermi level）移动到真空能级（Vacuum level）所需的最小能量。在催化研究中，功函数是表征材料电子结构的重要物理量，常用于分析表面反应性、电子迁移能力和催化选择性。

其数学表达式为：

其中：

  • ：真空区域的电势，即远离材料表面时电势趋于平坦的区域；
  • ：材料的费米能级；
  • ：功函数，单位通常为电子伏特（eV）。

# 步骤

在第一性原理计算中，功函数的计算一般基于slab模型,主要有以下四个步骤：

# 1. 结构优化

计算功函数之前先进行结构优化（和平常结构优化一样，准备好INCAR，KPOINTS,POSCAR,POTCAR和脚本文件），优化结束后收敛即可。

# 2. 静态计算

  • 静态计算需要先在当前目录建一个文件夹（比如：static），然后cp * static，将当前目录的文件都移动至static里面，cp CONTCAR POSCAR接着对static里面的INCAR参数进行修改，具体要改的是：
      IBRION = -1   # 不做结构优化
  NSW = 0       # 优化步数设为0（强制静态）
    1
    2
  • 设置非自洽电荷密度输出：
      LVTOT = .TRUE.   # 输出总静电势到 LOCPOT
  LVHAR = .TRUE.   # 输出赫特势（辅助分析）
    1
    2
  • 另外若slab非对称、上下表面不同，默认电势会发生偏移，此时需要Dipole修正，即在INCAR中添加：
      LDIPOL = .TRUE.   
  IDIPOL = 3           # z方向修正
  DIPOL = 0.5 0.5 0.5  # slab居中则取中点
    1
    2
    3
  • 修改好INCAR参数即可提交任务，等待计算完成进行下一步处理。
