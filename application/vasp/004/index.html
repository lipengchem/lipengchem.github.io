# VASP 能带与态密度计算完整流程指南

本文档记录了如何使用VASP进行能带（band structure）、态密度（DOS）与 Bader 电荷分析计算。

# 🔹 第一步：结构优化（Geometry Optimization）

# 关键输入文件：

  • POSCAR：待优化的初始结构
  • POTCAR：与 POSCAR 元素顺序一致
  • KPOINTS：适中的 K 点网格（如 6×6×6）
  • INCAR：结构优化参数
ENCUT = 520
EDIFF = 1E-5
EDIFFG = -0.03
ISIF = 3
IBRION = 2
NSW = 100
ISMEAR = -5
SIGMA = 0.05
LCHARG = .FALSE.
LWAVE  = .FALSE.
PREC   = Accurate

# 文件准备：

# 🔹 第三步：能带结构计算（Band Structure）

# 文件准备：

  • 首先复制SCF的INCAR、POSCAR、POTCAR、CHGCAR以及 vasp.script,使用vaspkit单独生成KPOINTS,修改INCAR参数
    ICHARG = 11
    LORBIT = 11
    NEDOS = 2000

# 🔹 第四步：态密度计算（Density of States）

# 文件准备：

  • 复制能带计算时设置的INCAR、POSCAR、POTCAR、vasp.script,复制SCF设置的CHGCAR，KPOINTS单独生成高密度网格（如 15×15×15），最后修改INCAR参数
    ISMEAR = -5
1

# 🔹 第五步：Bader 电荷分析（需 AECCAR）

chgsum.pl AECCAR0 AECCAR2
bader CHGCAR -ref CHGCAR_sum
