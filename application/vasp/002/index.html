# VASP 输入文件详解

VASP (opens new window)（Vienna Ab-initio Simulation Package）是一款广泛使用的第一性原理计算软件。在进行计算之前，用户需要准备四个主要的输入文件：

  • POSCAR：晶体结构文件
  • POTCAR：赝势文件
  • KPOINTS：k 点网格设置
  • INCAR：计算控制参数

本文将分别介绍它们的作用、格式与编写示例。

# 1. POSCAR—结构文件

作用：描述系统的晶体结构信息。

格式示例：

MoS2 monolayer                     # 自行命名该工作 
1.0                                # 缩放因子 
3.160000 0.000000 0.000000         # 第一晶格向量
-1.580000 2.736550 0.000000        # 第二晶格向量
0.000000 0.000000 20.000000        # 第三晶格向量 
Mo S                               # 元素种类，顺序需与 POTCAR 一致
1  2                               # 对应元素的原子个数
Selective Dynamics                 # 可选行，表示是否启用原子固定功能
Direct                             # 使用坐标类型:Direct(分数) 或 Cartesian(笛卡尔)
0.000000 0.000000 0.500000 T T T   # 原子坐标；T: 弛豫；F: 固定
0.333333 0.666667 0.500000 T T T
0.666667 0.333333 0.500000 T T T

# 2. POTCAR—赝势文件

作用：提供每种元素的赝势信息，用于描述电子与离子核之间的相互作用。

# 示例

PAW_PBE Mo 06Sep2000
  VRHFIN =Mo: s2 d4
  LEXCH  = PE
  ENMAX  = 293.228; ENMIN = 214.921
  ...
1
2
3
4
5
  • PAW_PBE Mo 06Sep2000：表示该赝势基于 PAW 方法和 PBE 泛函，为 Mo 元素生成于 2000 年 9 月 6 日。
  • VRHFIN =Mo: s2 d4：指价电子配置为 4d⁴ 5s²。
  • LEXCH = PE：表示使用 PBE 交换相关泛函。
  • ENMAX：推荐的最大截断能（单位 eV），应作为 INCAR 中 ENCUT 的参考值。

# 使用说明

  • POTCAR 文件通常由多个元素的赝势拼接而成。
  • 拼接顺序需与 POSCAR 中元素顺序严格一致。
  • 每个元素的赝势可以从 VASP 安装目录中的 potpaw_PBE、potpaw_LDA 等文件夹中获取。
  • 示例命令（拼接Mo和S的赝势）：
cat /path/to/potpaw_PBE/Mo/POTCAR /path/to/potpaw_PBE/S/POTCAR > POTCAR
1

# 3. KPOINTS—k点网格文件

作用：指定计算中用于布里渊区积分的 k 点采样方式，影响计算精度与效率。

# 常见写法：Monkhorst-Pack 网格（自动生成）

Automatic mesh
0
Monkhorst-Pack
6 6 1
0 0 0
1
2
3
4
5

# 字段说明

  • 第 1 行：注释，任意描述。
  • 第 2 行：0 表示自动生成网格。
  • 第 3 行：指定使用 Monkhorst-Pack 或 Gamma。
  • 第 4 行：k 点网格数量（x y z 方向）。
  • 第 5 行：偏移量，通常为 0 0 0。

# 其他写法：高对称路径撒点(常用于能带计算)

k-points along high-symmetry lines

0.5 0.0 0.0 ! X
0.5 0.5 0.0 ! M
0.0 0.0 0.0 ! Gamma

# 4. INCAR—计算参数控制文件

作用：控制计算任务的类型、精度、自洽过程等核心参数，下面总结了 VASP 中常见计算任务所使用的 INCAR 参数，供学习计算时参考。

# 结构优化示例

SYSTEM   = Structure Relaxation     # 任务名称（任意字符串）
ISTART   = 0                        # 是否读取波函数（0 = 初始化，1 = 读取 WAVECAR）
ICHARG   = 2                        # 电荷密度起始设置（2 = 初始化）
ENCUT    = 500                      # 平面波截断能（单位 eV，≥POTCAR 中最大 ENMAX）
PREC     = Accurate                 # 精度设置（Normal / Accurate / High）
ISIF     = 2                        # 优化内容（2 = 优化原子，3 = 原子+晶格+体积）
EDIFF    = 1E-4                     # 自洽能量收敛准则
EDIFFG   = -0.05                    # 力收敛准则（负号表示最大原子力阈值）
ISMEAR   = 1                        # 展宽方法（0 = 高斯，适合绝缘体）
SIGMA    = 0.1                      # 展宽宽度
ISPIN    = 2                        # 自旋设置（1 = 无自旋，2 = 自旋极化）
LREAL    = Auto                     # 实空间投影设置（Auto 推荐，大体系加速）
LWAVE    = .FALSE.                  # 是否输出 WAVECAR（波函数文件）
LCHARG   = .FALSE.                  # 是否输出 CHGCAR（电荷密度）
NSW      = 600                      # 最大离子移动步数
IBRION   = 2                        # 离子运动方式（2 = 共轭梯度）
POTIM    = 0.2~0.5                  # 离子步长系数（默认 0.5）
ISYM     = 0                        # 是否保留对称性（0 = 关闭）
ALGO     = Fast                     # 电子最小化算法（Fast 推荐用于结构优化）
NELM     = 100                      # 最大电子步数
VOSKOWN  = 0                        # 对应泛函类型：0 = PBE，1 = PW91（部分体系指定）
IVDW     = 12                       # 范德华修正11或12（DFT-D2/DFT-D3）
NPAR     = 4                        # 并行数，建议设置为节点数整除
MAGMOM  = N*X                       # 磁性体系设置，如 `MAGMOM = 4*0 1 1 -1`
LDAU    = .TRUE.                    # 启用 DFT+U

# 参数说明

参数 含义说明
SYSTEM 当前计算任务的描述文字
ENCUT 截断能（单位 eV），需不低于 POTCAR 中最大 ENMAX
PREC 精度设置（Normal / Accurate / High）
EDIFF 能量收敛准则，自洽迭代停止阈值
EDIFFG 力收敛准则（结构优化使用），负值表示最大残余力
ISMEAR 展宽方式：0 为高斯，-5 为 tetrahedron，1 为 Methfessel-Paxton
SIGMA 展宽参数，通常设置为 0.05（单位 eV）
IBRION 离子更新方式：
2 = 共轭梯度（结构优化）
5 = 振动频率计算
ISIF 控制优化内容（3 = 原子+晶格，2 = 原子，7 = 固定体积）
NSW 最大离子优化步数；振动计算应设为 1
NELM 最大电子步数（SCF 迭代）
LWAVE 是否输出 WAVECAR（波函数文件）
LCHARG 是否输出 CHGCAR（电荷密度文件）
ISTART 波函数起始方式（0 = 不读，1 = 读 WAVECAR）
ICHARG 电荷密度起始方式（0 = 初始化，1 = 读 CHGCAR，11 = 只读不 SCF）
ISPIN 是否启用自旋（1 = 非极化，2 = 极化）
LREAL 是否在实空间计算原子投影（Auto、.FALSE.）
ALGO 电子最小化算法（Fast / Normal / All）
ISYM 是否使用对称性（0 = 关闭）
LORBIT 是否输出投影态密度（10/11 推荐用于 DOS）
LVTOT 是否输出总电势（LOCPOT），用于功函数计算
LVHAR 输出哈特里电势（更光滑）
LAECHG 输出原子电荷（生成 AECCAR0/1 文件）
POTIM 离子移动步长（单位 Å）
振动计算推荐设置为 0.01~0.03
NPAR 并行参数，推荐为核数的因子（如 4、8）
AMIX 电荷密度混合参数（默认 0.2）
BMIX 电荷混合的预条件（默认 0.0001）
NSIM 并行 SCF 步数，用于金属体系的加速
VOSKOWN 赝势选择标记（0 = PBE，1 = PW91）
IVDW 范德华修正（DFT-D2:11，DFT-D3:12）
MAGMOM 初始磁矩设置，如 4*5 表示 4 个原子设置为 5 μB
LDAU 是否启用 DFT+U（.TRUE.）
LDAUTYPE DFT+U 类型（通常设为 2）
LDAUL 每个元素的 U 作用轨道（2 = d，-1 = 不加）
LDAUU 每个元素的 U 值
LDAUJ 每个元素的 J 值（如 0.8）
LCLIMB 是否启用 NEB 爬山图像（Climbing Image）
SPRING NEB 弹簧常数（负值表示自动设置）
IMAGES NEB 中插值图像数（中间点数）
NFREE 振动计算中用于有限差分的自由度（2 = 二阶差分）
NPLANE 振动计算加速选项之一（NPLANE = .TRUE. 时表示开启平面波优化）
